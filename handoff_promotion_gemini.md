# 🚀 Handoff Brief: Nesc'Advert AI Agent Template

Ce document contient toutes les informations stratégiques et techniques nécessaires pour travailler sur la **promotion** de l'Agent IA "Nesc'Advert" (Signals Dashboard Template).

> [!IMPORTANT]
> **Angle Marketing Principal** : Nous ne vendons pas une prestation de service, mais le **Template de l'Agent IA** lui-même (Vendre la "mine d'or").
> **Concept** : Outil d'automatisation de prospection "Agency-in-a-box" pour agences, freelances et équipes sales.

---

## 1. Identité du Produit
*   **Nom** : Nesc'Advert (Signals Dashboard)
*   **Positionnement** : Dashboard Intelligent de signaux d'achat (Social Selling Automation).
*   **Style Visuel** : Dark Premium, accents néons (#00ADEF), Glassmorphism.
*   **Cible** : Agences (SEO, Ads, Web), Sales Managers, Entrepreneurs IA.

## 2. La Proposition de Valeur (USP)
*   **Gain de temps massif** : Automatise 10h de recherche manuelle hebdo.
*   **Filtrage du "Bruit" à 95%** : L'IA ne garde que les prospects ayant une réelle intention d'achat.
*   **Réactivité Critique** : "First-Mover Advantage" sur LinkedIn.
*   **Personnalisation Automatique** : Génère des DM et commentaires pertinents.

## 3. Le "Secret Sauce" : Qualification IA (GPT-4o)
L'outil utilise un rôle de "Sales Auditor" ultra-conservateur.
*   **Scoring (0-100)** : Intent (60%) / Activity (40%).
*   **Expertise** : Filtrage automatique des concurrents et des posts "bruit" (job offers, personal updates).

## 4. Spécifications Techniques
*   **Frontend** : React / Vite / Lucide Icons.
*   **Backend** : Airtable (Base de données accessible au client).
*   **Automation** : Make.com Webhooks (Scraping LinkedIn/IG).
*   **Workflow** : Scan → AI Analysis → Human Validation → CRM Export.

## 5. Assets Déjà Créés
*   **Visuels LinkedIn** : Interface Dashboard + Scores.
*   **Script Vidéo IG** : Démo dynamique 9:16.
*   **Landing Page** : Captation de leads déjà intégrée au projet.

## 6. Pistes pour la Promotion
*   **"Agency in a box"** : Setup rapide pour scaler sa prospection.
*   **"Anti-Spam"** : Qualité vs quantité.
*   **"First Responder"** : Détecter avant tout le monde.
